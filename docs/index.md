# PyTransport 3.0

PyTransport is a numerical tool for the computation of primordial correlators in inflationary cosmology. It provides a Python interface to high-performance C++ routines.

## Core Features
* **Multi-Field Support**: Computes tree-level correlators for canonical and non-canonical field space.
* **Stability**: Integrates the Multi-Point Propagator (MPP) approach for enhanced numerical accuracy.
* **Efficiency**: Hybrid C++/Python design for fast computations.
